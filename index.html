The state of personal AI agents

Voice-first agents go mainstream

xAI’s Grok introduced a no-code Voice Agent Builder in beta, bundling telephony, tools, guardrails, and observability into a single stack. It’s aimed at high-volume production agents rather than hobby demos, with simple per‑minute pricing and fast setup.

Source: x.ai

Gemini and ChatGPT push toward agents

Google continues expanding Gemini’s agentic capabilities on desktop, while ChatGPT remains the reference point for general-purpose assistance. Both are evolving quickly — but most usage is still conversational or tool-based rather than durable computer operation.

Sources: engadget.com, techcrunch.com

Siri, Folk, Orchids as context

Siri remains a deeply embedded, voice-first assistant inside Apple’s ecosystem. Folk and Orchids sit more at the niche and experimental end of the agent spectrum — useful context for buyers, but not driving this week’s market momentum.

How the landscape is splitting

ChatGPT

World‑class general assistant for writing, reasoning, and planning. Strong for one‑off tasks and research, increasingly agentic — but repeated computer workflows still reset each run.

Gemini

Google’s aggressively evolving assistant, pushing browser‑native and desktop agent features. Powerful, but primarily optimized for Google’s ecosystem and stateless execution.

Grok

Opinionated, real‑time assistant. The new Grok Voice Agent Builder shows serious intent around production voice agents rather than personal computer automation.

Siri

Voice‑first, OS‑embedded, and improving — but still constrained in cross‑app computer operation compared to dedicated agents.

Folk & Orchids

Representative of niche and experimental approaches inside the broader agent market. Useful signals, not market leaders.

Super

Built for real computer work. Super agents operate browsers and desktops directly — and crucially reuse a computer-use cache, so repeated workflows get faster and cheaper over time instead of costing the same every run.

Why computer-use memory matters

Most personal AI agents still treat each task as a blank slate. When an agent can actually operate a computer — log in, navigate flows, handle edge cases — the missing piece becomes memory. Super’s reusable computer‑use cache means the second, tenth, and hundredth run of a workflow benefit from what the agent already learned.
